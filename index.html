Smart Irrigation System

Developed a remote-controlled smart irrigation system using a cloud server. The system is built with Arduino Mega, ESP32, and various sensors (soil moisture, rain, ultrasonic) and actuators (servo motors, electric pump, solenoid valves). A mobile application, created via RemoteXY.com, allows for global remote control, ensuring efficient water management.

Project 1: Remote Control of Water Treatment Station Using Siemens S7-1200 PLC

This project focused on remotely controlling a water treatment station by integrating the Siemens S7-1200 PLC with an OPC server and multiple client connections. The system was designed to operate in both automatic and manual modes, ensuring high reliability in industrial automation. The primary objectives included establishing a remote control setup, integrating the PLC with the OPC server, and deploying two UA Expert clients for testing, alongside a Python-based client for enhanced control. Key features included automatic and manual operation modes, with remote execution capabilities through UA Expert and Python clients. Technologies used included the Siemens S7-1200 PLC for reliable process control, OPC server for communication, UA Expert for system validation, and Python for advanced remote management. The project successfully demonstrated robust control over the water treatment process.

Project 2 :HMI interface for a water treatment

I developed an advanced HMI interface for a water treatment process, utilizing Vijeo Designer for interface design and Unity Pro for PLC programming. This project aimed to enhance user interaction and control over the water treatment operations by integrating a highly intuitive interface with real-time data monitoring and comprehensive control features. Key achievements included the creation of a user-friendly interface, effective real-time process visualization, and seamless integration with Unity Pro, ensuring reliable and efficient system operation. This project sharpened my expertise in HMI design and industrial process control, demonstrating my ability to integrate diverse technologies for optimized performance .

Project 3: Network Architecture for AVG Robot Installation using Modbus and TCP/IP

I completed a project focused on designing a network architecture for Automated Guided Vehicle (AVG) robots using Modbus and TCP/IP protocols. The project involved establishing a reliable client-server communication system, enabling remote control of the AVG robot through a TCP/IP interface and transmitting commands via RS232 Modbus. This solution ensured efficient and accurate control of the robot for automated material handling. The project significantly enhanced my expertise in industrial automation, network communication, and robotic control systems.
